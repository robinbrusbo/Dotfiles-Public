* Installation
#+begin_src sh
sudo pacman -S git
cd $HOME
git clone https://github.com/FlySlime/Dotfiles.git
cp -r Dotfiles/. .
rm -rf Dotfiles
chmod +x setup.sh
./setup.sh
#+end_src

* Configure
- ~/.doom.d/config.org
- ~/.xprofile
  - Change xrandr to suit your monitors
  - Change xinput to match your hardware e.g. your mouse.
- ~/.config/gtk-3.0/bookmarks
